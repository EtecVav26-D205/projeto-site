# ⚙️ Recursos do Site

## 📌 Funcionalidades Planejadas

O site contará com recursos modernos para melhorar a experiência do usuário e tornar a navegação mais prática e intuitiva.

---

# 🔐 Sistema de Login

O projeto pretende utilizar formulários de login e cadastro de usuários.

Os clientes poderão:
- criar contas
- realizar login
- acessar informações pessoais
- salvar dados para futuras compras

---

# 🗄️ Banco de Dados

O site utilizará banco de dados para armazenar informações importantes, como:

## 👤 Clientes
- nome
- e-mail
- senha
- telefone
- histórico de cadastro

---

## 👕 Produtos
- nome do produto
- categoria
- preço
- descrição
- imagens
- quantidade em estoque

---

# 🛒 Carrinho de Compras

O usuário poderá:
- adicionar produtos ao carrinho
- remover itens
- visualizar valores
- finalizar pedidos

---

# 🔎 Sistema de Pesquisa

O site contará com:
- barra de pesquisa
- filtros por categoria
- organização dos produtos

Facilitando a busca por roupas e acessórios.

---

# 🎨 Recursos Visuais

Para deixar o site mais moderno e interativo, pretendemos utilizar:
- animações
- efeito hover nos produtos
- carrossel de imagens
- banners promocionais
- menu responsivo

---

# 📱 Responsividade

O site será totalmente adaptado para:
- celulares
- tablets
- notebooks
- computadores

Garantindo boa experiência em diferentes tamanhos de tela.

---

# 📞 Área de Contato

O usuário poderá entrar em contato através de:
- formulário
- e-mail
- redes sociais
- telefone fictício da loja
