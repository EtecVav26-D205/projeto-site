# 🏙️ Maze — Streetwear

## 📌 Sobre o Projeto

Maze é um projeto de desenvolvimento de um site para uma marca de roupas streetwear.

A proposta do projeto é criar uma plataforma moderna, responsiva e visualmente atrativa, focada no público jovem que busca estilo, conforto e identidade visual através da moda urbana.

O site será desenvolvido utilizando conceitos de organização visual, experiência do usuário e Mobile First.

---

# 🎯 Objetivo

Desenvolver um site de roupas streetwear com:
- visual moderno
- navegação intuitiva
- responsividade
- organização prática dos produtos
- experiência agradável para o usuário

---

# 👥 Público-Alvo

O público principal da Maze são jovens e jovens adultos entre 18 e 35 anos que acompanham tendências de moda urbana e valorizam:
- autenticidade
- conforto
- praticidade
- estilo pessoal

---

# 📱 Mobile First

O projeto será desenvolvido utilizando o conceito Mobile First, priorizando inicialmente dispositivos móveis para garantir melhor experiência em celulares e tablets.

---

# ⚙️ Recursos Planejados

- sistema de login
- cadastro de usuários
- banco de dados
- catálogo de produtos
- carrinho de compras
- filtros de pesquisa
- menu responsivo
- animações e efeitos visuais
